# audit-leyu-lab

本仓库用于归档和发布多个独立的 HTML 页面。页面内容主要涉及信息核查与分析相关的资料整理，不针对任何特定的域名、网站或实体。

## 项目简介

`audit-leyu-lab` 是一个静态页面集合仓库。这里存放的每一个 HTML 文件都是一个独立的页面，旨在提供结构化的信息展示与存档功能。仓库本身不包含动态后端或数据库，所有内容均以纯静态 HTML 形式存在，便于直接浏览与分发。

## 目录结构

```
.
├── README.md           # 本文件
├── index.html          # 仓库首页 / 导航页面（可选）
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-example-1.html
│   ├── page-example-2.html
│   └── ...
└── assets/             # 公共资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：主要页面归档目录，每个文件对应一个独立主题或批次的内容。
- `assets/`：存放页面共享的样式表、脚本文件或图片素材，避免重复引用。

## 页面归档说明

- 每个 HTML 页面都是自包含的，可以单独打开浏览。
- 页面内容可能包含公开可获取的资料整理、分析笔记或数据可视化，不涉及任何非公开或敏感信息。
- 归档的页面可能不定期更新或补充，但不会对已有页面进行实质性修改（除非必要修正）。
- 页面内不会嵌入跟踪代码、广告或外部请求（除必要的字体、CDN 库外）。
